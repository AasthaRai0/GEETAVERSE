# GEETAVERSE

A responsive web application that displays inspirational quotes from the Bhagavad Gita with an engaging and interactive user experience. The project focuses on clean design, smooth animations, and accessibility across different devices.

## Features

* Display of motivational and spiritual Bhagavad Gita quotes.
* Smooth CSS transitions and animations for an enhanced user experience.
* Fully responsive design compatible with desktops, tablets, and mobile devices.
* Interactive and visually appealing user interface.

## Tech Stack

* HTML
* CSS
* JavaScript

## Project Highlights

* Implemented responsive layouts using modern CSS techniques.
* Added smooth transitions and hover effects to improve usability and engagement.
* Designed a simple and elegant interface to make reading quotes enjoyable.

## Run the Project

1. Clone the repository.
2. Open the project folder.
3. Run the `index.html` file in your browser.
